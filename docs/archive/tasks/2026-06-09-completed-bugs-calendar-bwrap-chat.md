### 2. bwrap 導入の結果、rp1:/~/.rustyclaw/ への書き込みができない

- **完了日**: 2026-06-09
- **内容**: `bwrap --ro-bind / /` でホスト全体が RO になり、Google カレンダー認証情報の書き込みエラー（Read-only file system）が発生。カレンダー情報が直接取得できない症状。
- **修正**: `--bind $HOME/.config $HOME/.config` 追加で RW バインドを確保し、認証情報書き込みを解消。
- **関連計画書**: -

### 3. Dashboard Chat にて「中断ボタン」の導入
