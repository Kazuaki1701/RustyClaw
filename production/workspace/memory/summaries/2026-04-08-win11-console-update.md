- **レンダリングの高速化:** Direct3D（Atlas）エンジンの採用により、テキスト表示やスクロールの遅延が大幅に削減されます。
- **Sixel画像表示のサポート:** ターミナル上で画像データを直接表示できるようになります。
- **機能の統合:** Windows Terminalで好評だった「正規表現検索」や「アクセシビリティ改善」が、標準のコマンドプロンプトでも利用可能になります。
