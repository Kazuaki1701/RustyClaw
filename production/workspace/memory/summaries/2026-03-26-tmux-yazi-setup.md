### 1. ペイン（画面分割）で常に横に置いておく
エディタとファイラーを並べて見たい場合に一番オーソドックスな方法です。
- **手順**: `Ctrl+b` `%` で画面を縦分割し、片方のペインでファイラー（`ranger`, `nnn`, `yazi` など）を起動します。
- **サイズ調整**: `Ctrl+b` を押したあと、**`Alt` を押しながら矢印キー** を押すと、ペインの幅を細かく調整できます。左端に細く常駐させるのが人気です。

### 2. 専用のウィンドウ（タブ）に配置する
画面は広く使いたいけれど、一瞬でファイラーに切り替えたい場合におすすめです。
- **手順**: `Ctrl+b` `c` で新しいウィンドウを作り、そこでファイラーを起動しておきます。
- **切り替え**: `Ctrl+b` `0`〜`9` （ウィンドウ番号）などでサッと行き来できます。

### 3. 【おすすめ】ポップアップで呼び出す（tmux 3.2以降）
現在作業している画面を崩さず、画面の中央にフワッとファイラーを表示させる、最近とても人気の設定です。

`~/.tmux.conf` に以下のようにショートカット（例: `Ctrl+b` のあと `f`）を登録します。
```tmux
# 例: Prefix + f で yazi (または ranger, lf など) を80%のサイズでポップアップ
bind f display-popup -E -w 80% -h 80% "yazi"
```
※設定を書き換えた後は `tmux source ~/.tmux.conf` で反映させてください。

yaziの導入、おめでとうございます！サクサク動いてとても快適なファイラーですよね✨

yaziでファイルを開くときのデフォルトエディタを `nano` に変更するには、主に **2つの方法** があります。システム全体で `nano` を標準にするのが一番簡単でおすすめです。

### 方法1：システム全体のデフォルトエディタにする（おすすめ）
yaziはシステムの環境変数 `$EDITOR` を参照します。お使いのシェル（bash や zsh など）の設定ファイルに追記するだけで変更できます。

`~/.bashrc` または `~/.zshrc` の末尾に以下の1行を追記してください。
```bash
export EDITOR="nano"
```
追記後、`source ~/.bashrc` （または `source ~/.zshrc`）を実行して設定を反映させれば完了です。

### 方法2：yaziの設定ファイルで直接指定する
「他のツールでは別のエディタを使いたいけれど、yaziの中だけは `nano` にしたい」という場合は、yaziの設定ファイル（`yazi.toml`）を編集します。

`~/.config/yazi/yazi.toml` を作成・編集し、`[opener]` セクションに以下のように設定します。
```toml
[opener]
edit = [
    { run = 'nano "$@"', block = true, desc = "nano" },
]
```

まずは一番簡単な「方法1」から試してみてくださいね😊
